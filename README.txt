Перед запуском программы, необходимо создать текстовый файл с входными данными

Исполнение программ производить на системе с 64-х битной архитектуре!!!
=================================================================================================================
________________________________________
Инсрукция к запуску на OS Windows:

Файл с входными данными должен находиться в той же директори, что и программа!

1. 	Открыть командную строку
2. 	С помощью команды (cd) перейти в директорию, 
	в которой находится программа
3.	Запуск программы имеет следующий формат: CftTest_win.exe input_file output_file key_data_type key_method_sort
	Значения аргументов:
		input_file - файл с исходными данными
		output_file - файл с выходными(отсортированными) данными
		key_data_type - тип данных для сортировки 
			("-s" для данных строкового типа; "-i" для данных целочисленного типа)
		key_method_sort - формат сортировки
			("-a" сортировка по возростанию; "-d" сортировка по убыванию)
	Пример команды для запуска программы в готовом варианте: 
		CftTest_win.exe in.txt out.txt -s -d
		(Сортировка данных файла будет произведена по строковому типу в порядке убывания)

Инсрукция к запуску на OS Linux:

Файл с входными данными должен находиться в той же директори, что и программа!

1. 	Открыть терминал
2. 	С помощью команды (cd) перейти в директорию, 
	в которой находится программа
3.	Программе присвоить флаг "x" для запуска, введя команду: chmod +x CftTest_linux
4.	Запуск программы имеет следующий формат: ./CftTest_linux input_file output_file key_data_type key_method_sort
	Значения аргументов:
		input_file - файл с исходными данными
		output_file - файл с выходными(отсортированными) данными
		key_data_type - тип данных для сортировки 
			("-s" для данных строкового типа; "-i" для данных целочисленного типа)
		key_method_sort - формат сортировки
			("-a" сортировка по возростанию; "-d" сортировка по убыванию)
	Пример команды для запуска программы в готовом варианте: 
		./CftTest_linux in.txt out.txt -s -d
		(Сортировка данных файла будет произведена по строковому типу в порядке убывания)
